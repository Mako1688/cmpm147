Experiment 3 - Alternate Worlds

Description

Through this expiriment I have thouroughly demenstrated use of tile mapping and generative algorithms that results in a coherent world whether it be a foggy dungeon or a breezy beach side village.

Technical

The text codes used in the grid are as follows: "-" for house "_" for sand "." for water. The code also randomizes certain tiles within those descriptions in order to give a sense of life to the image.

Reflection

I think this program came out decently well. I have been swamped with a number of stressful situations in my life while working on this project so I think being able to deal with all of that and getting this project done feels decently rewarding.
