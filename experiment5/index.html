Description

This is a generator that directly reads inspiring images, renders parameterized images, and adjusts its own designs to optimize an evaluation score. It mutates paramters in order to closer resemble the desired image.

Technical

  1. This project integrates 6 different inspiring images that are selectable from the drop-down on the display page. Each image highlights a potential stregnth and weakness of the program.
  2. The design object is initialized in inspiration-specific ways. Selecting different inspirations should lead to different behaviors of initDesign.
  3. The design is rendered in a way that can possibly produce results that feel related to the inspiring images. If we squint we are able to percieve the image.
  4. The mutation process makes incremental changes to the design. The parameters of design change based on the image but are always: x, y, fill, w, h, rotation, numPoints, and opacity. The rate parameter is implemented in a way that allows the user to almost stop exploration by sliding the rate down to 1%.

Reflection

I am very happy with how the generator turned out. I used many different methods of generation and as a result felt very excited to see how they would play out. This assignment really helped me understand the mutation process of images and shapes and I feel very good about using this method in the future.
